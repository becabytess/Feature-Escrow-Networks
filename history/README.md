# history — archived early FEN explorations

This folder holds **earlier, non-rigorous work** that predated the clean-slate track in [`fen_lab/`](../fen_lab/).

It is kept for reference only. **Do not treat results here as the project’s scientific claims.** The canonical narrative and evidence base live in the root [`README.md`](../README.md) and the experiments under `fen_lab/`.

| Path | What it was |
|------|-------------|
| `experiments/` | Early scripts (ablations, HAR, CIFAR-100, FordA bench, …) |
| `experiments2/` | MIT-BIH / FordA variant runs + logs |
| `experiments_h100/` | Larger sweeps (MLP, CNN, LSTM, transformer / FET) + CSV logs |
| `plots/` | Generated curves from those runs |
| `legacy_research_report.md` | Previous root write-up (broad claims; superseded) |

Session chat dumps may exist locally under `session_notes/` and are gitignored — not part of the public archive.
